# ORBIS-Hardware
Collection of 3D printable files, used for ORBIS projects.

## Housing

### Mobile

| Name | Description |
|---|---|
| Node 1 (a) | IP rated ORBIS Mesh enclosure (Raspberry Pi 5 + RAK Wireless LoRa) - 3-Pin Fly Wheel Twist Lock Battery Connector (MPU5, AN/PRC-148, AN/PRC-152, Battery 1) |
| Node 1 (b) | IP rated ORBIS Mesh enclosure (Raspberry Pi 5 + RAK Wireless LoRa) - proprietary battery (Battery 2) |
| Tracker 1 | ORBIS Mesh enclosure (Raspberry Pi Zero 2W) - built in battery |
| Gateway 1 | ORBIS Mesh enclosure with ATAK Server (Raspberry Pi 5 + RAK Wireless LoRa) - Connector for 12V external power supply (wall adapter, car cigarette lighter, ect.) |
| PTT 1 | IP rated PTT adapter for VoIP with integrated SPK and MIC, which also serves as a hub for Headsets, EUDs, USB Gadgets |

### Stationary

| Name | Description |
|---|---|
| HQ 1 | Stationary enclosure for ORBIS Mesh, ATAK Server, ATAK UI and LoRa (Raspberry Pi 5 + RAK Wireless LoRa). It features HDMI Ports, USB and headset / SPK-MIC connectors |

### Accesories

| Name | Description |
|---|---|
| Battery 1 | MPU5 style battery enclosure to fit 3S2P 18650 batteries |
| Battery 2 | Proprietary battery for Node 1 (b) |
| [GIZOisNT](GIZOisNT_Antenna/README.md) | Protective housing for antennas in the style of tactical gooseneck antennas |
